This data can be filtered. Add a query to the end of the URL, for example: http://localhost:5000/verbs/verb-display?end=i

The following filters are supported:

  • end: filters by the end of the verb root
  • start: filters by the start of the verb root
  • contains: filters for roots containing query
  • lv: (last vowel) filters for roots with the final vowel of the query

Filters can be stacked like this: ?lv=i&start=i&end=l&contains=l
